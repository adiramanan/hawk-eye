## Actions

- Expanded the v1 property gate to include the approved display and stroke-dash controls.

- Added multi-format color output support in the picker with Hex, RGB, HSL, and OKLCH serialization.

## Decisions

- V1 display hide/show maps to `display: none` versus restoring the prior non-none display mode when re-enabled.
- Solid fills preserve `backgroundColor` writes; gradients and images use shorthand background writes only when needed.
